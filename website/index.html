Two Ways to Run KetraTerm

Whether you need a dedicated shell terminal for command execution or a tool window inside a custom IDE, KetraTerm has you covered.

Standalone Desktop Application

Standalone Terminal

A fast, ready-to-run desktop terminal window. Includes session tab management, profile settings persistence, theme templates, and system PTY subprocess integration for Windows, Linux, and macOS.

A High-Fidelity Terminal Interface

🎨

Stunning SGR Graphic Styles

Experience 24-bit TrueColor RGB palettes. Features secondary underline decorations (single, double, curly, dotted, dashed) with custom color overrides.

🚀

Shell integration (OSC 133)

Visual command timeline rails identify prompt locations, highlight failed output rows with gutter dots, and report shell command exit codes directly.

🔗

Interactive Hyperlinks

OSC 8 hyperlink formatting detects URLs in stdout, rendering mouse-hover link lines and enabling direct Ctrl-click navigation.

🔔

Audible & Visual Bells

Never miss a command completion signal. Features regular audible warning tones and a subtle, non-intrusive edge-pulse visual flash overlay.

📁

Auto Directory Sync

OSC 7 tracks prompt working directories to dynamically update tab titles and enable quick "Open Terminal Here" shell splits.

😀

Native Emoji Rendering

Includes a built-in Segoe UI Emoji COLR/CPAL color glyph rasterizer on Windows, displaying emojis alongside system monospace fonts.

Simple TOML Settings

Tailor your workspace to match your style. KetraTerm reads settings from a standard, human-readable config.toml file stored in OS-standard app data directories.

Change your fonts, increase font-sizes, modify terminal column counts, configure cursor blinks, adjust underline line-height scaling, and select preconfigured themes like One Dark, Dracula, or Nord.

Graceful Error Recovery

If your configuration contains syntax errors, KetraTerm automatically falls back to safe system defaults instead of crashing.

# KetraTerm User Configuration Profile

[window]
columns = 100
rows = 30
scrollback_lines = 5000

[font]
family = "Cascadia Mono"
size = 14
line_height = 1.2
use_system_fallback_fonts = true

[theme]
name = "one-dark"

[behavior]
cursor_shape = "block"
cursor_blink_millis = 500
audible_bell = false
visual_bell = true
paste_sanitization = "normalize-line-endings"
