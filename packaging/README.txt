KetraTerm - High-Performance Desktop Terminal Emulator
======================================================

How to Launch:
- Windows: Run 'KetraTerm.exe' in this directory.
- Linux: Run 'bin/KetraTerm' in this directory.
- macOS: Double-click 'KetraTerm.app'.

Configuration:
KetraTerm is configured using a `config.toml` file.
For detailed guides, keyboard shortcuts, and configuration settings, visit:
https://ketraterm.github.io/
